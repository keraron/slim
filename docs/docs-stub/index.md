# Slim

**Slim (Secure Low-Latency Interactive Messaging)** is a communication framework
for secure, scalable messaging — the transport layer for protocols such as
[A2A](https://a2a.ai) and [MCP](https://modelcontextprotocol.io).

This site is the **repository documentation template** (MkDocs Material, AGNTCY
theme). Expand these pages with project-specific guides, or fold content in
from the [main Slim documentation on docs.agntcy.org](https://docs.agntcy.org/slim/overview/).
